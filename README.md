# A Sample FullStack Application

Front-End - Angular(version 11.1.1).
Back-End - Node-JS(Express.js).
Database - MongoDB.

**Features Include**
1.  login page 
2.  register page 
3.  home page.
    *  3.1 Sidebar
    *  3.2 Home page
	*  3.3 organization module
	*  3.4 Departments module
	*  3.5 Employees module
	*  3.6 Calender module
	*  3.7 Logout feature
